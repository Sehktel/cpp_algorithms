Добро пожаловать в пинг-понг сервер!

Это тестовое сообщение, которое будет передано клиенту сразу после установки 
TCP соединения с сервером.

Особенности реализации:
- Использование объектно-ориентированного подхода в C++
- Чтение настроек из файла settings.ini
- Автоматическая передача файла после подключения
- Игра в пинг-понг с клиентом

Информация о сервере:
- Протокол: TCP
- Тип: Синхронный сервер
- Платформа: Кроссплатформенный (Windows/Linux)
