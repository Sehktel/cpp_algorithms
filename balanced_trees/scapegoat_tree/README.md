# Scapegoat Tree

Scapegoat Tree - самобалансирующееся дерево, которое перестраивает поддерево при дисбалансе.

## Особенности

- **Балансировка**: Перестройка поддерева при превышении порога дисбаланса
- **Сложность**: O(log n) амортизированно
- **Преимущества**: Не требует parent-указателей, простая реализация
- **Недостатки**: Может быть медленнее при частых операциях

## Свойства

1. Использует параметр alpha (обычно 0.5 < alpha < 1)
2. Если дерево становится слишком несбалансированным, перестраивается поддерево
3. "Козел отпущения" (scapegoat) - узел, чье поддерево перестраивается

## Операции

- `insert(key)` - вставка
- `remove(key)` - удаление
- `search(key)` - поиск
- `rebuild(node)` - перестройка поддерева
