# Treap (Декартово дерево)

Treap - это комбинация бинарного дерева поиска и кучи, где каждому узлу присваивается случайный приоритет.

## Особенности

- **Балансировка**: Вероятностная, основана на случайных приоритетах
- **Сложность**: O(log n) в среднем
- **Преимущества**: Простая реализация, не требует parent-указателей
- **Недостатки**: Не гарантирует строгий баланс

## Свойства

1. По ключам: свойства бинарного дерева поиска
2. По приоритетам: свойства кучи (min-heap или max-heap)

## Операции

- `insert(key, priority)` - вставка с приоритетом
- `remove(key)` - удаление
- `search(key)` - поиск
- `split(key)` - разделение дерева
- `merge(t1, t2)` - объединение двух деревьев
