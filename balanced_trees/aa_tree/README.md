# AA Tree (Arne Andersson Tree)

AA Tree - это упрощенная версия красно-черного дерева, использующая уровни вместо цветов.

## Особенности

- **Балансировка**: Упрощенная версия RB-дерева
- **Сложность**: O(log n) для всех операций
- **Преимущества**: Проще реализация, чем RB-дерево, не требует parent-указателей
- **Недостатки**: Менее известен, чем RB-дерево

## Свойства

1. Каждый узел имеет уровень (level)
2. Листья имеют уровень 0
3. Если узел имеет уровень k, то его потомки имеют уровень k-1
4. Левый потомок не может иметь тот же уровень, что и родитель (skew)
5. Правый потомок не может иметь тот же уровень, что и правый внук (split)

## Операции

- `insert(key)` - вставка
- `remove(key)` - удаление
- `search(key)` - поиск
- `skew()` - операция выравнивания
- `split()` - операция разделения
